Uras e Prediais

Aplicativo mobile para técnicos de campo, com informações e localização de equipamentos da Oi no Rio de Janeiro

JavaScript React Native Android
Duração: 1 mes

Problema / Contexto

Os técnicos de campo terceirizados da Oi enfrentavam dificuldades para localizar e consultar informações de equipamentos de dados e voz (Uras, Urãos e Prediais) no Rio de Janeiro.

As planilhas e documentos existentes não eram práticas em ambientes externos e muitas vezes sem conectividade, tornando o trabalho lento e sujeito a erros.

Solução

Desenvolvi um aplicativo mobile em React Native que fornece acesso rápido e offline a todas as informações necessárias.

Criei uma interface minimalista com cores claras para facilitar a leitura sob luz intensa, implementei sistema de busca eficiente e navegação por categorias intuitiva.

Integrei também mapas do dispositivo para que os técnicos encontrem rapidamente os endereços dos equipamentos.

Destaques Técnicos

  • Desenvolvimento em React Native com JavaScript
  • Funcionalidade offline para acesso aos dados
  • Integração com mapas nativos do dispositivo
  • Navegação por categorias e sistema de busca eficiente
  • Design minimalista adaptado para ambientes com muita luz solar

Demonstração

Resultados

  • Facilitou o trabalho dos técnicos de campo, tornando o acesso à informação rápido e centralizado
  • Uso offline, excelente em áreas sem cobertura de internet
  • Interface intuitiva que reduz erros e melhora produtividade
  • Aplicativo leve e performático, adequado para aparelhos Android mais fracos

O que aprendi

  • Como otimizar aplicativos mobile para uso offline
  • Básico de UI para ambientes externos e alta luminosidade
  • Integração com mapas nativos usando React Native
  • Importância de fornecer acesso rápido e fácil a informações para profissionais de campo
